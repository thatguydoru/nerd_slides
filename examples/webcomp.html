Custom Elements

HTML elements you define yourself using Javascript. These inherits from the base HTMLElement class or any built-in elements.

js 
class YourMom extends HTMLElement {
    static observedAttributes = [];

    constructor() {
        super(); // Very important to call this first
    }

    // Lifecyle callbacks
    connectedCallback() {}
    disconnectedCallback() {}
    attributeChangedCallback(name, oldValue, newValue) {}

    // Other functionalities afterwards
}

customElements.define("your-mom", YourMom);
html 
<your-mom></your-mom>

Shadow realm DOM

Encapsulation of the style, markup, and behavior.

let shadow = this.attachShadow({ mode: "closed" })
or 
let shadow = this.attachShadow({ mode: "open" })
